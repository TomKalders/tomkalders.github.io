Conway's Game of Life in C++

Example of a glider cannon

About

Conway's Game of Life is a zero player game where you give a simulation an initial state, after which you can watch the simulation evolve. It uses cellular automata, a grid of cell with a state (in this case dead or alive), where the neighbours influence the state of the current cell. I'm currently using this for my research project and will be updating it soon to 3D.

Framework

The framework is a small framework made in C++. It uses SDL2 for rendering and user input. Apart from that I also use glm as a math library, mainly used for the vector class. SDL2 is linked dynamically and glm is linked statically.
