TechScape

Result of prototyping

About

TechScape is an escape room like puzzle game where you have to fix various puzzles which are similar to problems someone might have in a real life technician job. Colruyt Group is a giant retail chain in Belgium. The idea is to convince people (mainly people who just graduated secondary school, but not limited to) to join Colruyt Group's technical school to help with technical problems like in the Colruyt store.

Engine

The game is currently being developed in Unity and is being developed for mobile. We got out of prototyping phase and have now started production. This is the first mobile game project I have worked on.

Contribution

In the prototyping phase I prototyped:

  • Rotating a room with objects by swiping over the room
  • Moving between different rooms
  • An inventory system hooked up to a UI
  • A puzzle where you connect wires to activate/repair an object
  • A system that allows for puzzles to be activated/deactivated and have puzzle activate/deactivate other puzzles/objects

    • Currently in the production phase I have mainly worked on a helper script that captures touch input for mobile and calls a corresponding event like, OnTouchBegin or OnTouchMoved and will pass in what it hit and the touch info. This will help keep cleaner code throughout the project and will keep all the touch input handling in one script. It also helps with performance, due to not having multiple scripts performing a raycast when a finger has touched the screen to check if it hit an object.
